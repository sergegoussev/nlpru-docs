Check_word([word][, lemmatize=True][, remove_proper_nouns=True][, allow_acronyms=False][, exclude_english_words=True])¶

Check_word - checks if the word is good to be included for natural language analysis. It takes the specified word, and returns the a result dictionary.

Parameters

  • word – input the word to check

  • remove_proper_nouns –

  • allow_acronyms –

  • exclude_english_words – exclude english words from the Russian text

Praram lemmatize

Return type
